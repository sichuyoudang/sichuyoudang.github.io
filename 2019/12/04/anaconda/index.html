Anaconda

Posted on 2019-12-04

Anaconda 是一个用于科学计算的 Python发行版，它可以便捷获取包且对包能够进行管理，同时对环境进行统一管理。Anaconda 包含了 conda、Python 在内的超过180个科学包及其依赖项。

安装

从官网下载 Python 3.7 version 64-Bit Command-Line Installe 版本，下载完成后双击下载文件，一直点 “下一步” 直至完成安装。在终端中输入命令 conda list，如果显示已安装的包名和版本号，则安装成功。
