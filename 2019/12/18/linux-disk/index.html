Linux 查看磁盘使用情况

Posted on 2019-12-18

df

df 用于显示 Linux 系统中各文件系统的硬盘使用情况，包括文件系统所在硬盘分区的总容量、已使用的容量、剩余容量等。

du

du 查看单独目录点的使用情况，它使用递归方式总结磁盘使用情况，以获取目录及其子目录的大小。
